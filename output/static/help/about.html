RapydGlow is a tool which integrates GlowScript in order to use RapydScript instead of Javascript
Please, go and visit RapydScript and GlowScript to know more about them.
Not enough to use a nice language, RapydGlow allows operators overloading.
Jürg Lehni has exposed a nice way to do this on his blog Implementing Operator Overloading in JavaScript
Last addition, 'synchronous loop' with StreamLineJS from Bruno Jouhier
